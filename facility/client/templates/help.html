Help

Searching is provided among whole database. In that reason the some filters for search results are not available. The search results depends on many factors and results are mixed for search strings that matches values stored in database.

Search algorithm finds matches in following data in database:

  • Manager name
  • Manager email
  • Manager phone
  • Device name
  • Device serial number
  • Device category
  • Device Location:
    • Department
    • Faculty
    • Laboratory
  • Academical usage

We recommend to give as detailed as possible search parameters to match you exactly search. Otherwise you may recieve unexpectedly long set of results and filtering these results may be hard.
